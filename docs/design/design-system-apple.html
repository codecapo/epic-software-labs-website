Touch Target Requirements

HIG Requirement: 44×44pt Minimum

All interactive elements (buttons, links, form controls) must have a minimum touch target of 44×44 points. This ensures comfortable tapping on mobile devices and is a WCAG 2.1 Level AAA requirement.
